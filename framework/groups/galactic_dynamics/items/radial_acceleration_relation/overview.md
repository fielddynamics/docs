# Radial Acceleration Relation

The Radial Acceleration Relation (RAR) plots observed gravitational acceleration against baryonic gravitational acceleration for all measured radii across the SPARC galaxy sample.

## The relation

At each radius in each galaxy, two accelerations are computed:

- **g_bar**: the gravitational acceleration predicted from observed baryonic matter alone (Newtonian gravity applied to the photometric mass distribution)
- **g_obs**: the gravitational acceleration measured from the observed circular velocity (g_obs = v_obs^2 / r)

When plotted for all 175 SPARC galaxies (approximately 2,700 data points), these accelerations fall on a tight empirical relation rather than scattering randomly.

## Theoretical predictions

The GFD interpolating function mu(x) = x/(x+1) predicts a specific curve through the RAR plane. This can be compared against:

- The MOND simple interpolating function
- The Newtonian 1:1 line (g_obs = g_bar)
- The observed scatter

## Features

- All 175 SPARC galaxies plotted simultaneously
- Individual galaxy highlighting
- GFD, MOND, and Newtonian prediction curves
- Logarithmic axes with the characteristic acceleration scale a_0 marked
