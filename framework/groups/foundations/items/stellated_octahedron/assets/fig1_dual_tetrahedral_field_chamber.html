Dual Tetrahedral Field Chamber

Supplementary Figure 1

Construction

A stellated octahedron defined by two dual tetrahedra (TetA, TetB) inscribed in a cube. The field node has combinatorial parameter k = 4: four coupled faces, each partitioned into four sub-channels (3 spatial corners + 1 inverted core), giving 4 x 4 = 16 geometric channels.

Half-space inequalities

Four oriented planes partition local frame space:

  • L1 = x + y + z
  • L2 = x - y - z
  • L3 = -x + y - z
  • L4 = -x - y + z

TetA: L_i ≥ -1 for all i
TetB: L_i ≤ +1 for all i
Field Origin (octahedron): intersection TetA ∩ TetB

Colour key

  • Cyan -- TetA exposed faces (3 corners per face)
  • Gold -- Inverted core (TetB piercing TetA face)
  • Red -- TetB exposed faces

Resolution

This visualisation is not a fit to observational data. It renders the exact half-space boundaries that define the dual tetrahedral topology derived in the paper. Four million points are sampled uniformly and classified by the inequalities above.
